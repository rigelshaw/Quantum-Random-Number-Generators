Real-time CAPTCHA Delivery System

1️⃣

Arduino → Computer

Arduino generates a CAPTCHA and sends it through USB serial port

2️⃣

Node.js Bridge

Node.js reads serial data and prepares it for the webpage

3️⃣

Node.js → Website

Uses WebSockets to instantly push CAPTCHA to website

4️⃣

Website Receives Updates

Webpage listens for "captcha" messages and updates live

5️⃣

Manual Refresh Trigger

CAPTCHA appears only when Refresh clicked + Arduino sends

Real-time hardware-to-web pipeline with WebSocket communication and manual refresh synchronization
